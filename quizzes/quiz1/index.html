Quiz 1 Review

This first quiz will take approximately 20 minutes to complete. It will consist of a few multiple choice, as well as short answer questions. This will not involve any programming on the computer, and all answers will be handwritten. The quiz will cover material through Big-Oh notation and general concepts of linked lists.

The best way to prepare for this quiz is to review all labs as well as the first two homework assignments. In addition, notes that are posted on the course website are also helpful. Perhaps a good starting point is to review the in-class quiz we did on the first day of class.

Be familiar with the following topics:

- Static methods and static data.

- Inheritance including the relationship between parent and child classes.

- Interfaces and how they support polymorphism.

- Linked lists.

- Scope rules including public and private visibility.

- '==' vs 'equals()'

- Difference between O(1) and O(n)

This seems like quite a bit, but really it is an emphasis of what you have been programming the first month of class, and if you have been comfortable with the programming concepts, you should be reasonably-well prepared.
